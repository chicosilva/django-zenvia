Qual o prazo para meus créditos expirarem?

Os créditos não expiram, ficam acumulados para serem usados quando necessário.


Qual o tempo para os créditos de recargas serem disponibilizados?

Os créditos que utilizam a forma de pagamento "cartão de crédito", levam em média 5 minutos. Pagamentos efetuados por boleto bancário, levam até 3 dias úteis.


Quando um SMS é tarifado?

Uma mensagem só é tarifada quando o status é mostrado como "Mensagem Recebida"


Posso enviar qualquer tipo de SMS?

SMS com divulgação de qualquer tipo de ideologias, incluindo, mas não se limitando a, motivações políticas, canais de notícias, religiosas, correntes de pensamento, para uso de cunho pessoal, e outras.


Posso enviar SMS para campanhas políticas?

Não. É proibido envio de SMS com conteúdo eleitoral ou político.


Qual horário que posso enviar SMS?

SMS MARKETING só podem ser enviados no período de 09h00min até 21h00min


Posso enviar SMS todos os dias?

A cada 7 (sete) dias, um determinado USUÁRIO poderá ser impactado com no máximo 5 (cinco) DISPAROS de SMS.


O que é SMS Marketing?

É a SMS de texto que promove um serviço ou produto, para estimular a compra ou transação comercial com conotação publicitária, promocional ou de propaganda.


O que é SMS Corporativo?

É a mensagem de texto utilizada para comunicação estritamente sobre o serviço/produto prestado pelo Cliente, com cunho informativo, tais como cobrança, comunicações, alertas e demais conteúdos para notificações.
